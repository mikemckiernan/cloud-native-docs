Docker-in-Docker

You can also run GPU containers with Docker-in-Docker (dind). Just mount in the Docker socket to the container and then specify the CUDA container that you want to run:

$ sudo docker run -v /var/run/docker.sock:/var/run/docker.sock \
    docker run --rm --gpus all \
    nvidia/cuda:11.0-base \
    nvidia-smi
